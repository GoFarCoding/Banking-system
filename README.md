# Banking System

## Project Description
This project is a Java Banking System connected to a MySQL database using JDBC.

## Features
- Customer Registration
- Database Connectivity
- Console-Based Banking System

## Technologies Used
- Java
- MySQL
- JDBC
- GitHub

## Project Structure

```text
src/
 ├── main/
 ├── dao/
 ├── database/
 └── model/
```

## Database Setup
Run the SQL script:

```text
database/banking_system.sql
```
